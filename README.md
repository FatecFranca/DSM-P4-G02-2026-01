NeoVínculo - Monitoramento Neonatal IoT
O NeoVínculo é um projeto desenvolvido como Projeto Integrador (PI) para o curso de Desenvolvimento de Software Multiplataforma. O objetivo é monitorar em tempo real a temperatura e os batimentos cardíacos de bebês prematuros através de uma pulseira inteligente, enviando alertas imediatos para as mães e gerando dados estatísticos para a ONG Universo Prematuro.

Hardware: ESP32-C3 (Microcontrolador com Wi-Fi/Bluetooth).
Back-end: Node.js com Express.
Banco de Dados: PostgreSQL / MongoDB (Ainda em definição).
Comunicação: Protocolo HTTP / JSON.
